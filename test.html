Startanleitung für das SchulSystem

Zurück zur Startseite

So startest du das Programm auf deinem Windows-PC

  1. ZIP-Datei herunterladen:
    Lade dir die aktuelle Version von GitHub herunter:
    👉 Zum Download auf GitHub
  2. ZIP-Datei entpacken:
    Klicke mit der rechten Maustaste auf die ZIP-Datei und wähle „Alle extrahieren…”
    📂 Tipp: Entpacke sie z. B. auf den Desktop.
  3. Datei SchulSystem.exe starten:
    Öffne den entpackten Ordner und doppelklicke auf die Datei SchulSystem.exe
    🔐 Bei der ersten Ausführung kann eine Sicherheitsabfrage erscheinen – wähle "Trotzdem ausführen".
  4. Fertig!
    Das SchulSystem startet und zeigt dir das Hauptmenü. Beim ersten Start werden Beispieldaten geladen.

❗ Probleme beim Start?
Stelle sicher, dass du mindestens Windows 10 verwendest und keine Antivirensoftware die Ausführung blockiert.
